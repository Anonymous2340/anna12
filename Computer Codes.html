Explaintation:

The HTML < code > element defines a fragment of computer code.
The surrounded by < code > tags is typically displayed in the browser's default monospace font.

Sample Code:

< code >
x = 5;
y = 6;
z = x + y;
< code >

Result:

x = 5; y = 6, z = x + y;

The < code > element does not preserve extra whitespace and line breaks.
To fix this, you can put the < code > element inside a < span > element.
